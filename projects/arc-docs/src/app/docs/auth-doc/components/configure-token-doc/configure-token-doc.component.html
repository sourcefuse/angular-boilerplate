Setting a User token

When we get token we call authorise method, The authorize method is part of an authentication service. It is designed to handle the authorization process using a provided secret (such as an authorization code). This method initiates a command to obtain tokens, processes the response, and navigates the user based on the authorization result.

  • Process Response:If the response contains an access token and a refresh token: Save the tokens and their expiry. Navigate the user to the last accessed URL or the home path.Return true indicating successful authorization.
  • If tokens are not received, return false
