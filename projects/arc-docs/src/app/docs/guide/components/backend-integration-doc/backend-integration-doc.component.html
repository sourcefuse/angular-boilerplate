Overview

This guide provides the necessary steps to integrate a frontend application with a backend authentication service. It includes boilerplate code to ensure a smooth setup and usage of the authentication service without needing a backend developer.

Prerequisites
  • Node.js and npm installed
  • Angular CLI installed
  • PostgreSQL installed
  • Redis installed
  • Basic knowledge of Angular and REST APIs
