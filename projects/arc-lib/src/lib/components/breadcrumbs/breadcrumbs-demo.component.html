Breadcrumbs Component Examples

Flexible, dynamic, and static breadcrumbs for Angular projects.

Static Example

Use static breadcrumbs for simple, non-dynamic navigation trails. Icons, custom classes, and collapsing are supported.

Dynamic Example (User/Notes)

This example demonstrates dynamic breadcrumbs using route data and resolvers.
Click a button to see the breadcrumbs update with resolved user and note titles.

Current path: {{ currentPath }}
Resolved data: User: {{ resolvedUser.name }} | Note: {{ resolvedNote.title }}

How it works

  1. You navigate to a dynamic route (e.g., /user/12/notes/1).
  2. Resolvers fetch user and note data before the route activates.
  3. The breadcrumb function uses this data for labels.
  4. The breadcrumb trail updates automatically.

API

See all available inputs for customization:

Input Type Description
staticBreadcrumbs Breadcrumb[] Static breadcrumb items (optional)
separator string Separator character/string (default: /)
maxItems number Max breadcrumbs before collapsing (default: 8)
showIcons boolean Show icons if provided (default: false)
itemClass string Custom class for breadcrumb items
activeClass string Class for active (last) item
disabledClass string Class for disabled items
separatorClass string Class for separator
