3. Breadcrumb Inputs

See all available inputs for customization:

Input Description
staticBreadcrumbs Static breadcrumb items
separator Separator character/string
skipLink A flag to disable click navigation on a breadcrumb
resolver Angular service that pre-fetches data before routing
separatorClass Custom class applied to the separator between breadcrumb items
itemClass Custom class applied to each breadcrumb element.
